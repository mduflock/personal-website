1) This is a video of an AR drone flying around a classroom. (I would mute this before playing - the drone is pretty loud.) This was part of a group project that I did for my Intro to Network Security class. In addition to this basic controller, which controls yaw, pitch, roll, and allows the drone to land and take off, we also had to come up with attacks on other drones and defenses for our own. It was a lot of fun and I really enjoyed working with my group. (That's me laughing in the background, because it's always amazing when something you build makes something else work.) 3) This is an encryption project I did for my class on network security. It uses the Diffie-Helman method of creating a key using information sent in the open, and thus that both parties (and any eavesdroppers) know. It also sends a message and authenticates the signature that is attached to that message. This uses a basic messaging application that I set up for the class. Link to Project 3.
